# Discipline & Motivation App

## Description

Une application mobile visant à aider l'utilisateur à forger une discipline quotidienne, suivre ses routines et objectifs, et visualiser sa progression via un dashboard simple.

## Fonctionnalités - V1

### Objectifs

-   Créer un objectif (titre, description, date limite)
-   Voir la liste des objectifs
-   Marquer comme terminé / non terminé
-   Filtrer par “en cours” / “terminé”

### Routines

-   Créer une routine quotidienne ou hebdomadaire (ex. matin / soir)
-   Checkbox automatique pour cocher/décocher la tâche
-   Notifications ou rappels (optionnel MVP)

### Dashboard

-   Statistiques simples : nombre d’objectifs terminés, routines complétées, streaks
-   Graphique simple (barres ou cercles) pour visualiser la progression

### Widgets / Page d’accueil

-   Mini-widget avec routines à cocher rapidement
-   Mini-widget pour voir objectifs du jour

### Agenda

-   Liste des objectifs par jour / semaine
-   Possibilité de cocher depuis l’agenda

## Technologie

-   Flutter pour front et logique principale
-   Stockage local pour MVP (SQLite ou Hive)
-   GitHub pour versioning

## Priorités pour MVP

1. Gestion des objectifs (CRUD basique)
2. Gestion des routines (CRUD + checkbox automatique)
3. Dashboard statistique simple
4. Mini‑widgets sur la home page
5. Agenda simple lié aux objectifs et routines
